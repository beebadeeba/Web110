HTML and CSS Glossary

Section 2 - HTML

Tag
The opening or closing bracket that surrounds the term that is used on the webpage.
Element
The collection of the start tag, the content, and the end tag.
Attribute
More details that are added to a tag to do a specific job.
Self-Closing Elements
Elements that do not need a closing tag as they do not wrap around content.

Section 4 - Skeleton Code

Meta Viewport
The code that indicates the browser should not zoom, and should respect any responsive CSS code.
Doctype
The classification that tells the browser which version of HTML the document uses.
Title
What displays on the browser tab or in search listings on search engines.
Body
Element that holds all displayed content

Section 5 - Basic HTML Elements

Ordered Lists
the ol tag, by default displays items in a list numerically, can be customized.
Small
Represents text that is of little importance.
Emphasis
The em tag, adding italics to text.
Headings
Section titles in an HTML formatted document.
